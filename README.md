# House-Prices-Kaggle
This is the repository for the House Prices Kaggle competition

The competition link is found here: https://www.kaggle.com/competitions/house-prices-advanced-regression-techniques/overview

# Notes:

This project was coded hand-in-hand with DeepSeek. Full transparency of the chat log is available in the/DeepSeek Chat Logs folder.
